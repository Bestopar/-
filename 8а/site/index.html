jQuery - это JavaScript библиотека значительно упрощающая написание кода.

Девиз jQuery "write less, do more" (пиши меньше, делай больше) максимально точно отражает главное предназначение jQuery.

jQuery содержит функции позволяющие создавать анимацию и обработчики событий.

jQuery облегчает выбор элементов в DOM и создание AJAX запросов.
